# BOOK STORE APPLICATION

## BUILDING A MERN-BASED BOOK STORE MANAGEMENT SYSTEM

### BACKEND
#### ➡ The project leveraged MongoDB and Express.js to lay the backend foundation.
#### ➡ Mongoose acted as the bridge, connecting the application to the database and defining models for book titles, authors, and publication years.
#### ➡ These models were enriched with functions facilitating various database operations.


### FRONTEND
#### ➡ React components and pages were utilized to construct the user interface.
#### ➡ The project employs CRUD operations to facilitate data manipulation.
